Mejores prácticas de visualización de datos

Propósito: Use este auxiliar de trabajo para conocer las mejores prácticas de la visualización de datos.

Mejores prácticas para una comunicación clara

No importa qué método de comunicación use su organización, una comunicación clara y efectiva es clave. En lo que se refiere a la comunicación a través de la visualización de datos, deben seguirse varias mejores prácticas:

  • asegurarse de usar buenos datos
  • considerar si el propósito de la visualización de datos es declarativo o exploratorio
  • considerar la audiencia
  • usar múltiplos pequeños o interactividad al presentar muchos datos

Mejores prácticas para usar varios tipos de visualizaciones

Aunque los datos pueden presentarse técnicamente en cualquier tipo de gráfico, algunos gráficos son más adecuados para transmitir la información presentada por los datos.

Cuándo usar cada tipo de visualización de datos
Tipo de visualizaciónCuándo usarlo
Gráfico de líneasPara ilustrar cambios específicos con el tiempo
Gráfico de barrasPara hacer el seguimiento de cambios con el tiempo con el fin de realizar análisis comparativos entre un número limitado de variables
Diagrama de dispersiónPara ilustrar la relación o dependencia entre dos conjuntos de datos cuando existen muchos puntos de datos con los que trabajar
Gráfico circularPara mostrar proporciones y comparar pequeños grupos de información
Gráfico de mapaPara presentar visualizaciones basadas en datos geográficos
Coordenadas paralelasPara comparar una selección de productos con atributos idénticos

Mejores prácticas para diseñar visualizaciones de datos

El cerebro tiene una forma específica de interpretar lo que vemos o visualizamos. Para este fin, el uso de la teoría Gestalt puede convertir conceptos abstractos representados por datos en otros más tangibles.

Cómo usar cada ley Gestalt
Ley GestaltCómo usarla
Ley de PrägnanzSea simple y conciso, y organice los datos lógicamente. Use símbolos que tengan sentido inmediatamente sin requerir aclaración adicional.
Ley de continuidadAgrupe elementos que estén en consonancia entre sí; las líneas continuas hacen que los gráficos sean más legibles.
Ley de similitudAgrupe objetos de forma, color, tamaño, etc. similares.
Ley de proximidadAgrupe datos que apoyen el objetivo de un gráfico.

Mejores prácticas para evitar visualizaciones de datos imprecisas y engañosas

Para presentar una idea específica y mantener la credibilidad con el uso de la visualización de datos, deben usarse ciertas mejores prácticas clave:

  • empezar la escala en cero
  • no usar gráficos acumulativos
  • seguir convenciones gráficas estándar
  • no omitir datos innecesariamente
