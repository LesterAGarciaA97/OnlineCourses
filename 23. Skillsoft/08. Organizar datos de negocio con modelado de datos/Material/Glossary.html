Glosario: Organizar datos de negocio con modelado de datos

entidadConsulte entidad.
tipo de entidadTambién conocido como entidad, un grupo de objetos similares.
atributoUn elemento de datos que pertenece a un tipo de entidad. Por ejemplo, atributos para ubicaciones pueden incluir almacenes y oficinas.
modelo de datos conceptualUna visión de alto nivel de las necesidades de una organización.
modelo de datos lógicoUna descripción detallada de entidades y relaciones, normalmente en forma de tablas, en una base de datos.
modelo de datos físicoUn modelo derivado del modelo de datos conceptual. Incluye estructuras de tabla, columnas, claves foráneas y principales, así como relaciones.
desarrollo ágilNormalmente hace referencia al desarrollo de proyecto de software donde la priorización de varias tareas se reconfigura en función de las necesidades del cliente.
clave principalUna clave en una base de datos relacional que es específica de cada registro.
clave foráneaUna clave usada para representar relaciones entre filas en tablas en una base de datos relacional.
